\doxysubsection{Detailed Description}
Implementação mock de um fluxo para testes. 

A classe permite definir manualmente o valor retornado pela função {\ttfamily \doxylink{class_flow_mock_a0b34cfe64dd8db67455054f60521a343}{equation()}}, facilitando a validação do comportamento dos sistemas sem depender de cálculos reais.
