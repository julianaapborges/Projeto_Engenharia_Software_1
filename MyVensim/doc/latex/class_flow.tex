\doxysubsection{Detailed Description}
Classe base abstrata para fluxos (transferências entre sistemas). 


\begin{DoxyItemize}
\item Um fluxo conecta dois estoques (Source e Target) e transfere valores entre eles baseado em uma equação definida nas subclasses. 
\end{DoxyItemize}
