Repositório criado para a disciplina de Engenharia de Software I (BCC322), do Prof. Tiago Garcia de Senna Carneiro.\hypertarget{dir_ada5c03dfad4073e4e3629528318a961_autotoc_md1}{}\doxysubsection{\texorpdfstring{🚀 Sobre o Projeto}{🚀 Sobre o Projeto}}\label{dir_ada5c03dfad4073e4e3629528318a961_autotoc_md1}
O objetivo deste projeto é projetar, especificar (UML) e implementar uma API (framework) em C++ para a construção de simulações baseadas na {\bfseries{Dinâmica de Sistemas}}.

A API permitirá ao modelador criar "{}sistemas"{} (estoques de energia/matéria) e "{}fluxos"{} (que transportam essa energia/matéria), permitindo a simulação de modelos dinâmicos conforme a Teoria Geral de Sistemas. O loop de simulação principal seguirá o algoritmo geral descrito na Figura 5 do trabalho.\hypertarget{dir_ada5c03dfad4073e4e3629528318a961_autotoc_md2}{}\doxysubsubsection{\texorpdfstring{Funcionalidades (Casos de Uso)}{Funcionalidades (Casos de Uso)}}\label{dir_ada5c03dfad4073e4e3629528318a961_autotoc_md2}
O framework será desenvolvido utilizando TDD (Desenvolvimento Orientado por Testes) para implementar os seguintes modelos de validação\+:


\begin{DoxyItemize}
\item {\bfseries{Modelo 1\+:}} Crescimento Populacional Exponencial
\item {\bfseries{Modelo 2\+:}} Crescimento Populacional Logístico
\item {\bfseries{Modelo 3\+:}} Crescimento Populacional Complexo
